# Code of Conduct

## Purpose

This repository is a collaborative workspace. This Code of Conduct defines **expected behavior** to ensure productive, respectful, and professional interactions.

It complements existing organizational policies and applies to all interactions within this repository (code, pull requests, reviews, comments).

---

## Expected Behavior

All contributors are expected to:

* Communicate respectfully and professionally
* Provide constructive, actionable feedback
* Focus discussions on the work (code, design, requirements)
* Be open to feedback and alternative approaches
* Assume positive intent from others

---

## Unacceptable Behavior

The following behaviors are not tolerated:

* Personal attacks, harassment, or disrespectful language
* Dismissive or non-constructive feedback
* Discriminatory or inappropriate comments
* Escalation of conflicts in public discussions without attempt to resolve

---

## Collaboration Guidelines

* Keep pull requests focused and relevant
* Use clear, objective language in reviews
* Support feedback with rationale or references when possible
* Resolve disagreements through discussion, not authority
* When needed, involve a neutral reviewer or maintainer

---

## Responsibilities

### Contributors

* Follow repository standards and workflows
* Respond to feedback in a timely and professional manner

### Reviewers

* Provide fair, consistent, and constructive reviews
* Avoid blocking progress without clear justification

### Maintainers

* Ensure standards are applied consistently
* Facilitate resolution of conflicts
* Escalate issues when necessary

---

## Reporting and Escalation

If an issue cannot be resolved through normal collaboration:

1. Contact the repository maintainers
2. If required, escalate through appropriate organizational channels (e.g., team lead, manager, HR)

---

## Scope

This Code of Conduct applies to:

* Pull requests and code reviews
* Issues and discussions
* Any collaboration within this repository

---

## Disclaimer

This document supplements, but does not replace, existing organizational policies and codes of conduct.
